About StockWise

Welcome to the Stock Prediction System, an innovative web application designed to predict stock prices using advanced machine learning techniques. Our system leverages historical data and sophisticated models to forecast future stock prices, providing valuable insights to assist in making informed investment decisions.

Key Features

  • Predict Future Prices: Utilize LSTM networks to analyze historical stock data and predict future prices.
  • Data Integration: Fetch the latest stock data from Yahoo Finance for up-to-date predictions.
  • Visualization: View graphs comparing actual stock prices with predictions to assess model performance.
  • Model Management: Save and load trained models to optimize the prediction process without retraining.

How It Works

  1. Data Collection: Historical stock data is collected from a CSV file or fetched from Yahoo Finance.
  2. Model Training: If no model is available, the system trains an LSTM model on the historical data.
  3. Prediction: The trained model is used to predict future stock prices based on recent data.
  4. Visualization: The system generates and saves visualizations of actual versus predicted stock prices.

Technologies Used

  • Flask: A Python web framework used to build the web application.
  • TensorFlow/Keras: Libraries for building and training the LSTM neural network model.
  • yfinance: A library to fetch historical stock data from Yahoo Finance.
  • Matplotlib: A plotting library used to create visualizations of stock predictions and training results.
